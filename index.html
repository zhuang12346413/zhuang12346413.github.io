1.6 掩码

:::tip 如何理解掩码？
我们可以理解为掩码代表作用范围，限制某些方法的使用范围，或作用域
:::
:::tip opencv中掩码是什么？
掩码是一个8位的图像，也就是灰度图像、一通道的图像
:::
:::tip 掩码的规则是什么？
为0的位置不起作用，也就是黑色部分不起作用，拷贝的时候不拷贝黑色部分
:::
:::tip 如果想反转怎么办？
很简单判断0就可以了

2.2 指针遍历图像

:::tip
图像数据内存块大小，一个宽W高H彩色图像中，内存块的大小是WH3 unsigned char. 为了保持图像处理性能提高，但是会通过额外的像素来填补行的长度，比如行是8的整数倍。上面我们已经知道，长，宽，元素，通道的概念了，接下来，针对像素，我们可以了解到，像素大小，像素总和。
:::

:::note
这里我们注意到，图像的遍历，使用了两次循环，如果我们在处理每一个图像的时都去遍历两次，这样会很不方便。为了简化，我们多半都是使用一维数组来做计算，然后再将图像还原成原本大小。
:::
